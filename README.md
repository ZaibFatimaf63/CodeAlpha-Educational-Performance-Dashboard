# Educational Performance and Resource Allocation Dashboard

## Objective
Developed a Power BI dashboard to evaluate student performance and support resource allocation decisions in educational institutions.

## Features
- Student performance analysis by gender
- Impact of test preparation on scores
- Parent education level analysis
- Performance category distribution
- Interactive filtering

## Tools Used
- Microsoft Power BI
- Student Performance Dataset

## Insights
- Students completing test preparation courses perform better.
- Female students show slightly higher average performance.
- Higher parental education is associated with improved student outcomes.
- Medium and low performing groups can be prioritized for additional educational support.
